# Analyse Statistique Marketing - Python

Ce projet a été réalisé dans un cadre scolaire, dans le module 
"Statistiques Marketing", à l'aide de Jupyter Notebook.

## Contexte
L'analyse porte sur les données de l'entreprise **Livrenet**, 
une librairie en ligne, à partir de 3 fichiers CSV :
- Informations sur les clients
- Informations sur les produits vendus
- Informations sur les transactions

## Objectifs
- Réaliser une analyse exploratoire (EDA) de chaque fichier
- Identifier et corriger les incohérences dans les données
- Analyser les ventes et en extraire des insights

## Résultats
Les analyses et recommandations marketing sont disponibles 
dans le fichier `Livrenet - Stats Marketing.pdf`.

## Outils utilisés
- Python / Jupyter Notebook
- Pandas, Matplotlib, Seaborn
